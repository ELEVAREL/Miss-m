EKB Consumer Decision Model — 5 Stages

1. Need Recognition — Internal or external stimulus triggers awareness of a want/need
2. Information Search — Consumer seeks options (internal memory + external sources)
3. Evaluation of Alternatives — Weighs options using decision criteria (price, brand, quality)
4. Purchase Decision — Can be disrupted by social influence or unexpected circumstances
5. Post-Purchase Evaluation — Satisfaction vs expectations → cognitive dissonance if gap exists

Cognitive Dissonance: Buyer's regret. Marketers reduce this via reassurance marketing (follow-up emails, loyalty programs, money-back guarantees).
